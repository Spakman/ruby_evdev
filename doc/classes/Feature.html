activated?()

Returns true if the feature is currently activated, false otherwise. This query is only supported on KEY,LED, and SND feature_types. This function will raise if the feature_type is not one of those three.

name()

Returns the name of the feature. Check out CodeMappings for the available names

  EventDevice.open(ARGV[0] , "a+") { |device|
    led = device.feature_type_named('LED').feature_named('SCROLLL')
    led.write_value(1)
  }

Writes a value to the device for the given feature type and feature. For LEDs writing 0/1 turns them on and off. Exactly how the device responds to this writing is up to the device driver writer.
