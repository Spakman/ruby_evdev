Class FeatureType
In: feature.rb
Parent: Object

an object corresponding to one type of input/output for a device. All leds, buttons, keys etc. are grouped together into a FeatureType. This type object can be queried to determine what features it supports.

Public Instance methods

activated_features()

Returns a list of all the Features within this type that are currently activated (that is, on for leds, depressed for keys, playing for sounds). This is only supported for the KEY LED and SND feature types. It will raise for other types.

feature_named(name)

Returns the Feature within this type for the given name (check out CodeMappings for the list of possible names). This will throw if the Feature is not supported by this device (even if it’s one of the possible values for the FeatureType)

name()

Returns the name of the feature type. Check out CodeMappings for the available names

supported_features()

Returns the list of Feature objects that this FeatureType supports
